# СТРАТЕГИЧЕСКИЕ НАПРАВЛЕНИЯ РАЗВИТИЯ (BOOST)

Этот документ описывает 4 вектора для качественного скачка системы Trade League.

## 1. Сентимент-Арбитраж (News Front-running)
Переход от анализа "прошлых новостей" к стриминговому анализу Gemini Flash. Цель — вход в сделку в первые 10-30 секунд после выхода новости.

## 2. Матрица Состояний Рынка (Microstructure)
Замена SMA50 на динамическую классификацию (Trend / Mean Reversion / Panic / Range). Адаптация активности агентов под конкретную фазу.

## 3. Векторная Память (Long-term Experience)
Интеграция векторной БД для хранения успешных паттернов. Позволяет ИИ сравнивать текущую ситуацию с историческими прецедентами без раздувания контекста.

## 4. Прунинг Контекста (Feature Selection)
Математический отбор индикаторов для каждого агента. Удаление избыточных данных, не влияющих на PnL, для снижения затрат на токены и повышения точности сигналов.
