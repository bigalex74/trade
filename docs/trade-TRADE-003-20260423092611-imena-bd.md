# Имена баз данных торговой системы

Дата: 2026-04-23

Ветка изменения: `TRADE-003`

Версия проекта после изменения: `0.1.2`

## Назначение

Эта справка фиксирует имена PostgreSQL-баз, которые используются торговыми скриптами.

Документ нужен для восстановления операционного контекста, проверки логов, ручных SQL-запросов и диагностики отчетов.

## Биржевые трейдеры

Основная PostgreSQL-база для биржевых трейдеров:

```text
market_research
```

Торговые таблицы находятся в схеме:

```text
trading
```

Ключевые таблицы:

```text
trading.portfolio
trading.position
trading.journal
```

Типовые полные имена:

```text
market_research.trading.portfolio
market_research.trading.position
market_research.trading.journal
```

Пример подключения:

```bash
psql -h localhost -U n8n_user -d market_research
```

## Криптотрейдеры

Отдельная PostgreSQL-база для криптотрейдеров:

```text
crypto_research
```

Криптотрейдеры также используют схему:

```text
trading
```

Типовые полные имена:

```text
crypto_research.trading.portfolio
crypto_research.trading.position
crypto_research.trading.journal
```

Пример подключения:

```bash
psql -h localhost -U n8n_user -d crypto_research
```

## Важно

Не путать с БД n8n:

```text
n8n_database
```

`n8n_database` относится к n8n-инфраструктуре и не является основной торговой БД трейдеров.

## Источник в коде

Биржевой трейдер использует переменную окружения `DB_NAME`, а дефолт в коде:

```text
market_research
```

Криптотрейдер использует переменную окружения `DB_NAME_CRYPTO`, а дефолт в коде:

```text
crypto_research
```

Пользователь БД по умолчанию:

```text
n8n_user
```

Хост по умолчанию:

```text
localhost
```
