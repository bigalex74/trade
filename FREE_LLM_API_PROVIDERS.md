# СПИСОК БЕСПЛАТНЫХ И ТРИАЛЬНЫХ LLM API ДЛЯ FALLBACK-КАСКАДА

Поскольку бесплатные тарифы имеют жесткие лимиты (Rate Limits), лучшая стратегия отказоустойчивости — это **каскадная маршрутизация**. Если провайдер A возвращает ошибку 429 (Too Many Requests), система автоматически пробует провайдера B.

Ниже представлен список сервисов, предоставляющих доступ к мощным открытым моделям (Llama 3, Mixtral, Qwen) бесплатно или с хорошими стартовыми кредитами.

## 1. Сверхбыстрые провайдеры (Лучшая замена Ollama)

### [Groq](https://console.groq.com/)
- **Модели:** Llama 3 (8B/70B), Mixtral 8x7B.
- **Скорость:** Феноменальная (сотни токенов в секунду, специализированное железо LPU).
- **Условия:** Полностью бесплатный (на данный момент), но есть жесткие лимиты на количество запросов в минуту (RPM) и токенов в минуту (TPM). Отлично подходит для редких, но быстрых запросов.

### [Together AI](https://api.together.xyz/)
- **Модели:** Огромный выбор (Llama 3, Qwen, Mistral и десятки других).
- **Скорость:** Очень высокая.
- **Условия:** Дают $5 приветственного кредита при регистрации. Модели стоят доли цента за миллион токенов, поэтому $5 хватит на месяцы тестирования и разработки.

### [DeepInfra](https://deepinfra.com/)
- **Модели:** Llama 3, Mixtral, Qwen.
- **Условия:** Часто предоставляют небольшие бесплатные лимиты или триальные кредиты. Отличаются очень низкими ценами при переходе на платный тариф.

## 2. Агрегаторы и платформы для разработчиков

### [OpenRouter](https://openrouter.ai/)
- **Модели:** Агрегатор вообще всех существующих моделей (включая платные GPT-4 и Claude).
- **Условия:** У них есть специальный раздел **"Free Models"**. Они предоставляют бесплатный доступ к некоторым открытым моделям (например, Llama 3 8B, Gemma) через свой единый API. Это самый удобный вариант, так как код пишется под один API, а модели можно менять.

### [GitHub Models (Azure AI)](https://github.com/marketplace/models)
- **Модели:** GPT-4o-mini, Llama 3, Mistral, Cohere.
- **Условия:** Если у вас есть аккаунт GitHub, вы можете бесплатно использовать их API для экспериментов (Personal Access Token). Лимиты строгие (например, 15-50 запросов в минуту), но для Fallback-режима этого достаточно. Мощнейший и очень надежный инструмент от Microsoft.

### [Hugging Face Serverless Inference API](https://huggingface.co/inference-api)
- **Модели:** Более 100,000 моделей, размещенных на Hugging Face.
- **Условия:** Бесплатно для тестирования и прототипирования. Лимиты динамические и зависят от загруженности серверов HF. Менее надежно для продакшена, но отлично для резерва.

## 3. Провайдеры собственных моделей

### [Mistral AI (La Plateforme)](https://console.mistral.ai/)
- **Модели:** Mistral NeMo, Mixtral, Mistral Large (платная).
- **Условия:** У них есть бесплатный тариф (Free Tier) с доступом к их моделям с ограничениями по RPM.

### [Cohere](https://dashboard.cohere.com/)
- **Модели:** Command R, Command R+ (отлично подходят для RAG и работы с документами).
- **Условия:** Предоставляют бесплатный API-ключ для обучения и прототипирования (Trial Key) с лимитами (около 100 запросов в минуту для Command R).

## 🏆 РЕКОМЕНДУЕМАЯ СТРАТЕГИЯ КАСКАДА (LiteLLM)

Чтобы не писать отдельный код для каждого из этих провайдеров, рекомендуется использовать библиотеку **`litellm`** (Python). Она позволяет вызывать любой API (Groq, Together, OpenRouter, Gemini) через стандартный формат OpenAI.

**Пример идеального Fallback-каскада:**
1. **Primary:** Google Gemini Flash (бесплатный тариф, большие лимиты).
2. **Fallback 1:** Groq (Llama 3 70B) — если Gemini завис, получаем ответ мгновенно.
3. **Fallback 2:** GitHub Models (Llama 3) — если у Groq кончились лимиты.
4. **Fallback 3:** OpenRouter (Free Models) — последний рубеж защиты.
