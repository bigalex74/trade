# Схема входящих сообщений Telegram (n8n)

При импорте документа или отправке сообщения n8n получает JSON следующей структуры:

## Импорт документа
- `message_id`: ID сообщения
- `chat.id`: ID пользователя
- `document.file_id`: ID файла для скачивания через Telegram API
- `document.file_name`: Имя файла
- `document.mime_type`: Тип файла

## Текстовое сообщение
- `message_id`: ID сообщения
- `chat.id`: ID пользователя
- `text`: Текст сообщения

Используйте узел **If** с условием `{{ !!$json.message.document }}` для маршрутизации потоков.
